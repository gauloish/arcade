# Arcade

Arduino Arcade Emulator

## Games

- Pong
- Snake
- Tetris
- Space Invaders

## Components

All components is listed below:

```
- [1] Arduino Uno
- [1] Display MAX7219
- [1] Display TM1637
- [4] Potentiometer
- [4] Push Button
- [4] Resistor 1k ohms
- [-] Jumpers
```
